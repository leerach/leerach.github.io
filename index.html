3D SF Eviction Visualization Model

Constructed 3D geospatial visualizations of eviction notices to analyze communities within San Francisco most impacted by evictions by integrating Deck.gl and NodeJS while using public data.

Enrollment Graph

Redesigned entire client-side visual representation of enrollment graph for Berkeley Law courses in JavaScript and HTML5 using Chart.js to create an aesthetic and user interactive interface for multiline charts.

Lines of Action Board Game

Implemented the functionality of the manual player and utilized game trees, heuristics, and minimax algorithms to create an efficient AI to beat its opponent. Based on the board game Lines of Action by Claude Soucie.

Engima Machine

Recreated the German Enigma machine used in WWII to encrypt and decode messages, programming the permutations, substitution ciphers, and configurations of each interchangeable component of the system.

Gitlet: Recreation of Git

Created from scratch a working implementation of the git version-control system starting from the detailed design documentation to structure each component and replicate staging files, commits, etc.

Autocorrected Typing Software

Wrote a program that measures typing speed and implements autocorrect with data abstraction, tree recursion, and containers.

Girls in Engineering Group Lead Volunteer

Autonomous Object Detection

Optimized color and shape detection in phone camera by incorporating open source computer vision software (OpenCV) to enhance real-time image processing and communication with external sensors.

Scheme Interpreter

Built a scheme interpreter in Python that reads Scheme expressions, evaluates them, and prints the results.
